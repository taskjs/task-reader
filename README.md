# task-reader
> Read files.

## The "reader" task

### Usage Examples

```js
var reader = new (require('task-reader'))
reader.run(inputs, options, logger)
```

### Options

#### options.paths
Type: `array`

Files path.

#### options.encoding
Type: `string`

Files encoding.
